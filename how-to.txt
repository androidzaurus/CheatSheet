how-to
-----------------------------------
1) Login to http://github.com through browser
2) Create repository
3) Follow the instruction

$ mkdir CheatSheet
$ cd CheatSheet
$ git init
$ touch README
$ git add README
$ git commit -m 'first commit'
$ git remote add origin git@github.com:androidzaurus/CheatSheet.git
$ git push origin master
